# Anfal Sacco Mobile Releases

This repository is used to manage and distribute official versions of the Anfal Sacco mobile application using GitHub Releases.

## 📦 Purpose

This repository serves as a central place to:

* Publish new versions of the Dayax mobile app
* Upload app binaries (e.g. APK files)
* Provide release notes for each version
* Generate stable download links for users

These download links are used within the app for update prompts and can also be shared directly with users.

---

## 🚀 How It Works

For every new version of the app:

1. A new **Release** is created in this repository
2. The app binary (APK) is uploaded as a release asset
3. Release notes are added to describe the changes
4. The generated download link is used:

   * Inside the app (for update notifications)
   * Or shared with users

---

## 🔖 Versioning

We follow semantic versioning:

vMAJOR.MINOR.PATCH

Examples:

* v1.0.0 → Initial release
* v1.1.0 → New features and improvements
* v1.1.1 → Bug fixes

---

## 📝 Release Notes

Each release should include:

* New features
* Improvements
* Bug fixes
* Any important user-facing changes

---

## 🔗 Usage

Release assets provide direct download links that are used for:

* In-app update prompts
* Manual installation by users

---

## ⚠️ Notes

* Only production-ready builds should be published
* Do not delete old releases (used for version history and updates)
* Ensure version names and release notes are clear and consistent
